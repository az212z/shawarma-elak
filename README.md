# شاورما إيلاك — موقع مطعم (Shawarma Elak)

موقع تعريفي فاخر لمطعم **شاورما إيلاك** في جدة. شاورما عربي على الفحم (لحم ودجاج، صحون وسندويتش). موقع ثابت (HTML/CSS/JS فقط) بدون أي build step، جاهز للنشر على GitHub Pages.

## الرابط المباشر
- متوقّع: `https://az212z.github.io/shawarma-elak/`

## المعلومات
- **النشاط:** مطعم شاورما عربي — لحم ودجاج، صحون وسندويتش، إضافات وصوصات، مقبّلات.
- **المدينة:** جدة.
- **التقييم:** 4.3 على خرائط قوقل (531 تقييم) — مذكور كما هو من المصدر.
- **الجوال / الاتصال:** 0533324298
- **واتساب:** 966533324298
- **الموقع:** [خرائط قوقل](https://www.google.com/maps/search/?api=1&query=Shawarma%20Elak%20Jeddah)

## المميّزات
- موشن توقيعي: **سيخ شاورما يدور** ببطء (إحساس 3D عبر scaleX wobble)، وهج جمر، دخان خفيف، وشرائح تتقشّر وتتساقط على الطبق. مع fallback كامل لـ `prefers-reduced-motion` يُظهر الحالة النهائية ساكنة.
- Ken-burns للهيرو على صورة حقيقية، سكرول-ريفيل، تكبير عند المرور على الكروت، تصغير الهيدر عند السكرول، لمعة CTA.
- قائمة جوال ملء الشاشة (100vw/100dvh) بخلفية صلبة وزر X واضح.
- نموذج طلب → يفتح واتساب برسالة عربية جاهزة + حفظ محلي تجريبي (localStorage) + توست نجاح.
- أزرار عائمة: واتساب + اتصال + خرائط.
- RTL، نصوص سعودية محايدة جندريًا، أيقونات SVG، تباين ≥ 4.5:1، focus-visible، aria-labels.
- JSON-LD لـ Restaurant مع التقييم الحقيقي و `servesCuisine: Middle Eastern`.

## الـ Backend
لا يوجد backend. النموذج يعمل عبر `wa.me` (رابط واتساب) + `localStorage` للعرض التجريبي فقط؛ لا يُرسل أي بيانات لخادم.

صور حقيقية للمطعم (من ملف العميل/خرائط قوقل). تمّت **تنقيتها**: استُبعدت لوحات الأسعار والصور الضبابية والواجهة المصوّرة من داخل السيارة، وأُبقيت 7 صور (واجهة، أطباق استوديو نظيفة، جلسات داخلية).
